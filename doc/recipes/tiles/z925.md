# tiles/z925: Geopotential 925hPa

Mixer: **default**

## Inputs

* **z925**:
    * Model **cosmo**:
        * **Arkimet matcher**: `product:GRIB1,,2,6; level:GRIB1,100,925`
        * **grib_filter matcher**: `shortName is "z" and indicatorOfTypeOfLevel == 100 and level == 925 and centre != 98`
    * Model **ifs**:
        * **Arkimet matcher**: `product:GRIB1,98,128,156; level:GRIB1,100,925`
        * **grib_filter matcher**: `shortName is "gh" and indicatorOfTypeOfLevel == 100 and level == 925 and centre == 98`
        * **mgrib {k}**: `False`
        * **mgrib {k}**: `0.1`
